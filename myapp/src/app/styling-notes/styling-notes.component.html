Intro

Bootstrap is a CSS Framework for developing responsive and mobile-first websites.

CSS

To learn more about CSS, you can visit the mozilla docs here, and view some guides and cheatsheets here.

Selectors

Use selectors to set CSS styles to something like this: selector: { property: value; }

There is a good reference for selectors here but the big ones are:

  • .myclass for elements with class="myclass"
  • #myid for elements with id="myid"
  • element for elements like p, div, etc.
  • * for all elements

Box model

On any webpage, the browser renders every element as a rectangular box according to the CSS basic box model.

You can read more about the box model from the mozilla docs here. But basically, every element has a content area, a padding area, a border area, and a margin area, each of which can be adjusted.

Flex

Flex is a CSS module that tries to efficiently distribute items and space within a container, even when their sizes are unknown and/or dynamic. Flexbox is typically used in components of an application, or for small scale layouts. You can also learn more at this flexbox cheatsheet.

Grid

Grid is a CSS module that helps in designing two-dimentional grid based UIs. It is good for designing overall page layouts. You can also learn more at this grid cheatsheet.
